# Демо для жюри — MedServicePrice.kz

## Запуск (2 терминала)

**Терминал 1 — Backend**
```bash
cd backend
pip install -r requirements.txt
# БД уже в репозитории: medtech.db (или python reset_and_load.py для полного скрапинга)
python -m uvicorn app.main:app --reload --port 8000
```

**Терминал 2 — Frontend**
```bash
cd frontend
npm install
npm run dev
```

Откройте: **http://localhost:3000**

> Frontend проксирует `/api/*` на `http://127.0.0.1:8000` — без бэкенда поиск не работает.

## Сценарий демо (3–5 минут)

### 1. Главная — поиск как Aviasales
1. Выберите город (12 городов: Астана, Алматы, Шымкент, Караганда…)
2. Введите в поиск: **«Общий анализ крови»** или **«ОАК»**
3. Нажмите «Сравнить цены»

### 2. Сравнение цен
- Режимы: **Список** · **Таблица** · **Карта** (Leaflet)
- Сортировка: по цене, рейтингу, названию
- Фильтры: цена, рейтинг, верификация
- На карточке — **дата актуальности прайса**
- Клик по названию клиники → **карточка клиники** со всем прайсом

### 3. Примеры запросов для жюри

| Услуга | Город | Что показать |
|--------|-------|--------------|
| Общий анализ крови | Алматы | INVITRO vs HELIX vs KDL |
| Приём: кардиолог | Астана | DOQ.kz — агрегатор клиник |
| УЗИ брюшной полости | Алматы | HELIX диагностика |
| SVF терапия | Астана | MEDELICA (medelica.kz) |

### 4. Техническая часть (если спросят)

**Источники (28 скраперов):**
- invitro.kz, helix.kz, kdl.kz / kdlolymp.kz
- doq.kz (API, 12 городов)
- medelica.kz, aksay.kaznmu.edu.kz (PDF)
- olymp.kz → kdlolymp.kz

**API документация:** http://127.0.0.1:8000/docs

**Метрики:** http://127.0.0.1:8000/api/stats

**Скрапинг вручную:**
```bash
curl -X POST "http://127.0.0.1:8000/api/scrape/run"
curl http://127.0.0.1:8000/api/scrape/sources
```

**Raw-слой / нормализация:**
```bash
curl http://127.0.0.1:8000/api/normalize/unmatched
```

### 5. Что сказать про ограничения
- **Онлайн-запись** — демо-UI: кнопка «Записаться» открывает форму заявки
- **mck.kz** — домен недоступен, ошибка логируется, остальные источники работают
- **medel.kz** — паркуется; данные с **medelica.kz** (МЕДЭЛ, Астана)
- **aksai-clinic.kz** — DNS нет; данные с **aksay.kaznmu.edu.kz**

## Быстрый скрапинг (без полного цикла)

```bash
cd backend
python scrape_and_ingest.py --source invitro_kz --source helix_kz --source doq_almaty
```

Полный скрапинг по всем зарегистрированным источникам занимает **30–60 минут**.
